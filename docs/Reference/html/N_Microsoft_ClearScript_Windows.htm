Microsoft.ClearScript.Windows Namespace

The Microsoft.ClearScript.Windows namespace contains types that provide access to Windows Script engines such as JScript and VBScript in a desktop environment.
Classes
 ClassDescription
JScriptEngine Represents an instance of the JScript engine in a desktop environment.
Nothing Represents an empty object reference.
VBScriptEngine Represents an instance of the VBScript engine in a desktop environment.
WindowsScriptEngine Provides the base implementation for all Windows Script engines in a desktop environment.
Interfaces
 InterfaceDescription
IHostWindow Allows Windows Script engines to display dialogs within the host's user interface.
IWindowsScriptObject Represents a Windows Script object.
Enumerations
 EnumerationDescription
WindowsScriptEngineFlags Defines options for initializing a new Windows Script engine instance.
